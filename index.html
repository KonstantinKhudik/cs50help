CS50 Courses on edX

Need quick access to the lectures and projects for your CS50 course on edX? Here are some quick links to help you out.

Introductory

CS50T (cs50.edx.org/technology) - CS50's Understanding Technology

This is CS50’s introduction to technology for students who don’t (yet!) consider themselves computer persons.

CS50B (cs50.edx.org/business) - CS50's Computer Science for Business Professionals

This is CS50’s introduction to computer science for business professionals, designed for managers, product managers, founders, and decision-makers more generally.

CS50x (cs50.edx.org) - CS50's Introduction to Computer Science

An introduction to the intellectual enterprises of computer science and the art of programming.

CS50AP (cs50.edx.org/ap) - CS50's AP® Computer Science Principles

This is CS50 AP, Harvard University's introduction to the intellectual enterprises of computer science and the art of programming for students in high school, which satisfies the College Board's new AP CS Principles curriculum framework.

Intermediate

CS50W (cs50.edx.org/web) - CS50's Web Programming with Python and JavaScript

This course picks up where CS50 leaves off, diving more deeply into the design and implementation of web apps with Python, JavaScript, and SQL using frameworks like Flask, Django, and Bootstrap.

CS50G (cs50.edx.org/games) - CS50's Introduction to Game Development

Learn about the development of 2D and 3D interactive games in this hands-on course, as you explore the design of games such as Super Mario Bros., Pokémon, Angry Birds, and more.

CS50M (cs50.edx.org/mobile) - CS50's Mobile App Development with React Native

Learn about mobile app development with React Native, a popular framework maintained by Facebook that enables cross-platform native apps using JavaScript without Java or Swift.
